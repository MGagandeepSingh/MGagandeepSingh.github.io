Java's Primitive Data Type

Updated: 21-Dec-2022
Author: Gagandeep Singh

Java is a strongly typed language, which means each data type such as integers, characters, decimals is predefined as part of the programming language and every variable should be defined with one of the type.

Java has following 8 primitive data types. All of them are signed values except for boolean and char which are java's unsigned values.

# Data Type Size Values Default
1 byte 1 byte -128 to 127 0
2 short 2 byte -32768 to 32767 0
3 int 4 byte -2,14,74,83,648 to 2,14,74,83,647 0
4 long 8 byte 92,23,37,20,36,85,47,75,808 to 92,23,37,20,36,85,47,75,807 0L
5 float 4 byte Stores fractional numbers. Sufficient for storing 6 to 7 decimal digits 0.0f
6 double 8 byte Stores fractional numbers. Sufficient for storing 15 decimal digits 0.0d
7 boolean 1 byte Stores true or false values false
8 char 2 byte Stored a single character / letter or ASCII values '\u0000'
